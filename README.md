# shi-calendar项目

> 一个自定义的日历组件（A custom small calendar）

### 项目安装

``` bash
npm install
npm run dev
```

# npm安装使用：shi-calendar

``` bash
npm i shi-calendar

import shiCalendar from 'shi-calendar'

<shi-calendar></shi-calendar>
```
